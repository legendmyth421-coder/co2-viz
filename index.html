About This Visualization

Question

Which countries drove global CO2 emissions growth from 1990 to 2023, and what does the projected trajectory look like to 2030? This dashboard lets users interactively explore emission trends for 11 major economies across multiple coordinated views.

Design Rationale

The main view uses a multi-line time-series chart with gradient fills. Historical data (solid lines) is visually separated from the 2024–2030 forecast (dashed lines) by a vertical separator. A stacked area mode reveals cumulative totals. Three metric modes — absolute, per capita, and GDP intensity — let users explore different dimensions of the same data. A coordinated bar chart, donut share chart, and choropleth world map with a color-scale legend all update together as the scrubber year changes. Region grouping aggregates by Americas / Asia / Europe for a higher-level view. Country comparison mode shows side-by-side stat cards for any two countries. Double-clicking a country toggle opens a floating fact card with key statistics.

Interaction Techniques

Implemented interactions: (1) country toggles with flags; (2) draggable year scrubber syncing all views; (3) animated playback with speed control; (4) D3 brush for time zoom; (5) three metric toggles (absolute / per capita / intensity); (6) line vs. stacked area chart type; (7) region grouping; (8) compare mode with stat cards; (9) hover highlighting + tooltip; (10) clickable event annotations; (11) map hover tooltip; (12) map color-scale legend; (13) country fact cards on double-click; (14) cumulative emissions counter; (15) PNG export.

Data Sources

CO2 emissions: World Bank (EN.ATM.CO2E.KT). Population: World Bank (SP.POP.TOTL). GDP: World Bank (NY.GDP.MKTP.CD). World map: Natural Earth via world-atlas@2. Forecast: linear extrapolation from 2020–2023 trend.
